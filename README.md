# FATED_polymarket
Polymarket Project - analysis of market efficiency on crypto markets, comparison with traditional derivatives.


# Available data 


- **deribit_full.parquet** contains full historical data on Deribit BTC Calls/Puts from Nov2025 to April 2026

- **market_summary.parquet** contains summary for all individual markets on daily BTC price range questions

- **prices.parquet** contains histroical *yes* and *no* prices for daily BTC price range questions

- **trades.parquet** contains executed trades for daily BTC price range questions
